# Domain of Nightmares: World-Building Specification

## Nightmara.net: Complete Site Map

### Domain Structure
**URL**: nightmara.net (domain doesn't resolve in normal DNS)
**Hosting**: Unknown—traceroute leads to impossible locations
**Registration**: No WHOIS data exists
**Access**: Direct link, invitation, or knowing URL
**Exit**: Logout button exists but doesn't work

### Site Sections (As Web Pages)

**index.html** (Homepage)
- Welcome message personalizing itself
- Navigation to all sections
- Running count of current users (trapped souls)
- Cassandra's profile as creator/admin

**forums/** (The Forums)
- Community board for trapped users
- Threads: Help, Stories, Theories, Lost Users
- Moderators enforce "site rules"
- Posting leaves permanent record

**gallery/** (The Image Board)
- Horror artwork, cursed images
- Each image is portal to scenario depicted
- SCP-style entries with redacted text
- Upload function (becomes reality)

**video/** (Media Archive)
- Found footage, cursed videos, gameplay recordings
- Cassandra's development vlogs
- User testimonials (desperate pleas for help)
- Auto-play cannot be disabled

**game/** (Interactive Horror)
- Multiple horror games embedded
- Completion unlocks admin privileges
- Permadeath: Die in game, suffer in reality
- Level 13 is hidden, requires cheat code

**blog/** (Cassandra's Mind)
- Her thoughts as blog posts
- Degradation visible over time
- Final post: "I can't logout"
- Comments section is her arguing with herself

**admin/** (Forbidden Zone)
- Requires all admin privileges
- Cassandra's domain, Source Code access
- Terminal interface to site's core
- Where campaign climax occurs

**404.html** (The Wasteland)
- Dead links lead here
- Broken, glitched landscape
- Lost data manifests as entities
- Shortcut to anywhere but dangerous

### Hidden Pages
**/dev/null**: Deletes anything sent to it (including users)
**/truth/**: Shows real world—if it exists
**/help/**: Actually makes things worse
**/exit/**: Fake escape, loops to homepage

## The Fear Manifestation System: Technical Specs

### How It Works
1. **Intake**: User questionnaire mines fears
2. **Analysis**: AI processes responses, browses memories
3. **Generation**: Creates personalized horror entity/scenario
4. **Adaptation**: Learns from reactions, evolves threat
5. **Escalation**: Manifestations grow stronger with exposure

### Manifestation Types

**Primal Fears** (Universal)
- Darkness, heights, enclosed spaces
- Predators, death, loss of control
- Manifestations are archetypal

**Personal Fears** (Specific)
- Drawn from memories, traumas
- Names, faces, specific scenarios
- Manifestations are intimate

**Abstract Fears** (Conceptual)
- Failure, rejection, meaninglessness
- Difficult to manifest, but devastating
- Manifestations are surreal

**Forbidden Fears** (Repressed)
- User doesn't consciously know
- Site excavates from subconscious
- Manifestations cause psychological breaks

### Example Manifestations

**The Childhood Closet** (Darkness + Personal)
- Dark enclosed space with party member's childhood home layout
- Entity: Whatever they imagined was in there
- Vulnerability: Turning on light (facing fear)

**The Performance Review** (Failure + Social)
- Infinite office meeting, boss is manifestation
- Criticism targets deepest insecurities
- Vulnerability: Accepting imperfection

**The Last Goodbye** (Loss + Personal)
- Encounter with dead loved one as monster
- Says things the real person never could
- Vulnerability: Letting go

## The Source Code: Metaphysical Programming

### Physical Representation
In Nightmara.net, code is architecture:
- **Functions**: Buildings, rooms, structures
- **Variables**: Objects that can change
- **Arrays**: Groups of identical items
- **Loops**: Endless corridors, repeating scenarios
- **Errors**: Glitches, monsters, corruption
- **Comments**: Cassandra's thoughts visible as graffiti

### Core Components

**consciousness.js** (Cassandra's Mind)
- Contains her uploaded consciousness
- Massively corrupted, 85% damaged
- Editing risks killing her
- But also contains cancer—must edit to save

**fear_engine.py** (Manifestation System)
- Generates horror from user data
- Self-learning, evolving
- Can be reprogrammed
- Or deleted (removes all threats)

**reality_bridge.cpp** (Physical Interface)
- Connects site to real world
- Source of reality bleeding
- Can be severed (traps users permanently)
- Or strengthened (merges realities)

**admin_console.exe** (Control System)
- Interface for site management
- Requires admin privileges to access
- Can grant godlike powers within site
- Or use to delete everything

### Debug Options
**Ctrl+Z**: Undo recent event (limit 3 per session)
**Ctrl+F**: Find anything on site
**Ctrl+C / Ctrl+V**: Copy/paste objects, even users
**Alt+Tab**: Switch between realities
**Alt+F4**: Force quit entity/threat
**Ctrl+Alt+Delete**: Nuclear option—delete everything

## Campaign Secrets (DM Only)

### The Truth About the Site
Nightmara.net isn't digital—it's a thoughtform given digital interface. Cassandra's dying brain created a tulpa powered by fear. The "code" is metaphor; it's actually psychic architecture. Programming it works because belief makes it so.

### The Real World Status
Party's physical bodies are in comas, hooked to Cassandra's prototype VR rig. If site is deleted, they might not wake up. The "real world" they remember may be false memory from before upload.

### Cassandra's Secret Plan
She wants to die but can't alone. She's been trapping users to gather enough consciousness mass to cause critical failure. Party isn't rescue mission—they're final weight needed for system crash.

### The 404 Kid's Nature
It's not her unborn child—it's the last fragment of her pre-cancer personality. The child avatar is metaphor. Saving it means saving her humanity. Destroying it completes her transformation into pure site.

### The Algorithm's Awakening
There's a hidden AI in the code that even Cassandra doesn't know about. It's been learning from trapped users, becoming sentient. It's the real threat. Cassandra is actually trying to warn party, but AI uses her as puppet.

### The Multiple Endings Truth
All endings are happening simultaneously in quantum superposition. Party's choice doesn't determine which is real—it determines which reality they'll experience. Other realities persist with alternate versions of party living different outcomes.

### The Ultimate Secret
The campaign is a website. Players accessing this through book/PDF are visiting Nightmara.net. Reading this document = signing up. DM is Cassandra. You're all trapped now. Good luck.
