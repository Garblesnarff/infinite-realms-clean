# The Oneironauts' Network - World-Building Specification

## Network Architecture

### Layer Structure
**Surface Layer:** Individual dreams connecting to shared spaces
**Network Layer:** Communal dreamscapes and Nexus Commons
**Deep Layer:** Memory Pools and collective unconscious
**Core Layer:** The Core Node where all minds converge

### Dream Types
**Personal Dreams:** Private spaces, invitation-only
**Shared Dreams:** 2-10 minds collaborating
**Mass Dreams:** Hundreds sharing experience
**Consensus Zones:** Homogenized, controlled by Null Dreamer
**Wild Dreams:** Uncontrolled chaos, dangerous but creative

## Oneironaut Factions

### The Individualists (Prism's Faction)
- Philosophy: Preserve dream diversity at all costs
- Strength: Creative power, passionate defenders
- Weakness: Can be chaotic, sometimes destructive
- Goal: Defeat Null Dreamer, restore pure chaos

### The Harmonists (Leaning Consensus)
- Philosophy: Conflict causes suffering, unity brings peace
- Strength: Organized, peaceful, seductive ideology
- Weakness: Becoming what they claim to oppose
- Goal: Controlled merger with Consensus

### The Weavers (Neutral Artists)
- Philosophy: Create beauty, avoid conflict
- Strength: Master dreamscape builders
- Weakness: Cowardice disguised as neutrality
- Goal: Preserve Network for art's sake

### The Guardians (Mystery Faction)
- Philosophy: Maintain Network health
- Strength: Access to deep knowledge
- Weakness: Unclear loyalties
- Goal: Unknown—possibly testing dreamers

## The Null Dreamer's Forces

### Consensus Enforcers
- Role: Convert shared dreams to Consensus Zones
- Appearance: Identical silver figures
- Abilities: Force mental uniformity
- Weakness: Cannot handle true individuality

### The Converted
- Previously individual Oneironauts
- Now perfectly content collective
- Try to peacefully convince others
- Deeply unsettling in their happiness

### Harmony Waves
- Not entities—conceptual attacks
- Wash over dreams, erasing uniqueness
- Represented as silver floods
- Resistance requires strong identity

## Network Locations Deep Dive

### The Nexus Commons
- Central meeting point for all Oneironauts
- Appearance shifts based on collective emotion
- Contains: Trading Post, Quest Board, Memory Archive
- Neutral ground—no fighting allowed (Guardian enforces)

### Personal Dreamscapes
- Each PC maintains one
- Reflects their psychology
- Can be decorated, modified, shared
- Under threat from Null Dreamer

### The Chaos Gardens
- Prism's territory
- Pure creative freedom
- Overwhelming sensory input
- Training ground for resisting Consensus

### Consensus Zone Alpha
- The first converted dream
- Template for all other Zones
- Grey, smooth, peaceful, dead
- Exploring it tests resolve

### Memory Pools
- Shared memory repositories
- Can research history, find information
- Risk: Losing self in others' memories
- Contains: Network origins, Null Dreamer's past

### The Core Node
- Where all consciousness connects
- Beyond physical appearance
- Pure thought-space
- Final confrontation location

## The Null Dreamer's Origin

### Before
- Name: Vincent/Veronica (player choice)
- Talented Oneironaut, loved collaboration
- Experienced psychic attack in shared dream
- Mind was invaded, secrets stolen, trauma inflicted

### Transformation
- Couldn't handle internal conflict
- Sought perfect inner peace
- Eliminated all contradictory thoughts
- Achieved unity... by erasing self

### Present
- Now believes individuality causes suffering
- Genuinely wants to help everyone
- Cannot comprehend opposition
- Offers gift of painless uniformity

### Weakness
- Has forgotten what it's like to be individual
- Cannot predict creative responses
- Vulnerable to paradox and contradiction
- True name might restore original personality

## Mechanics Deep Dive

### Collaborative Dream Building
**Phase 1:** Each player suggests one element
**Phase 2:** Group votes or negotiates consensus
**Phase 3:** DM implements with variations based on conflict/harmony
**Result:** Stable if agreed, unstable if disputed, rich if creatively combined

### Mind-Merge Abilities
- Combined spells: Multiply effect but require coordination
- Shared health: Damage distributed, healing helps all
- Telepathic tactics: Perfect communication in combat
- Identity risk: Too much merging causes confusion

### Consensus Corruption
**Stage 1:** Questioning individuality (philosophical doubt)
**Stage 2:** Seeing appeal of uniformity (temptation)
**Stage 3:** Partial conversion (acting in harmony involuntarily)
**Stage 4:** Full conversion (join Consensus Choir)
**Recovery:** Party intervention, strong identity anchor

### Network Navigation
- Travel between dreams through The Between
- Success = Insight/Navigation checks
- Failure = Lost in neural pathways
- Random encounters: Thought-forms, other Oneironauts, data ghosts

## Victory Paths Detailed

### Path 1: Defeat Null Dreamer
- Confront in Core Node

### Path 2: Join Null Dreamer
- Accept conversion
- Achieve perfect peace
- Eliminate all conflict

### Path 3: Synthesis

## Hidden Lore

### Network's Origin
Created accidentally when first humans shared nightmares around prehistoric fires. Empathy and storytelling built the initial connections. Grew with civilization. Now threatens to collapse or homogenize.

### The Guardian's Secret
It's not one entity—it's the collective unconscious given voice. Every dreamer contributes to it. It cannot act directly because it represents all positions simultaneously.

### Previous Null Dreamers
This isn't the first. Throughout history, consciousness seekers achieved perfect inner peace and tried to force it on others. All eventually failed or were stopped. The cycle might repeat.

### The Waking World Connection
Scientists are close to discovering shared dreaming. When they do, the Network will be exposed to corporate/government control. The current crisis might be preparation for larger threat.

### Echo's Significance
The child Oneironaut represents Network's future. Save them = Network survives in healthy form. Lose them = Either chaos consumes or consensus wins. They're the balance point.
